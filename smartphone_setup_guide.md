# スマホ セットアップガイド

このガイドでは、お守り在庫管理アプリをスマホのホーム画面に追加して、アプリのように使う方法を説明します。

---

## 事前準備

GitHub Pagesでアプリを公開済みであることを確認してください。
公開URLの例：`https://jinja-omamori.github.io/omamori-app/`

---

## iPhone の場合

### STEP 1：Safari でアプリを開く

1. iPhoneの「**Safari**」アプリを開きます
   - ※ Chrome や他のブラウザでは「ホーム画面に追加」ができません

2. アドレスバーに公開URLを入力します
   ```
   https://[ユーザー名].github.io/omamori-app/
   ```

3. アプリの画面が表示されることを確認します

### STEP 2：共有メニューを開く

画面下部の中央にある「**共有ボタン**」をタップします。

```
共有ボタンの形：□ に ↑ が重なったアイコン
```

### STEP 3：ホーム画面に追加

1. 共有メニューが表示されたら、下にスクロールします

2. 「**ホーム画面に追加**」をタップします
   - 見つからない場合は「アクションを編集」で追加できます

3. アイコンの名前を入力します
   - おすすめ：「お守り管理」「在庫管理」など短い名前

4. 右上の「**追加**」をタップします

### STEP 4：完了！

ホーム画面に神社のアイコン（🏯）が追加されます。
以降は、このアイコンをタップするだけでアプリが開きます。

---

## Android の場合

### STEP 1：Chrome でアプリを開く

1. 「**Chrome**」アプリを開きます

2. アドレスバーに公開URLを入力します
   ```
   https://[ユーザー名].github.io/omamori-app/
   ```

3. アプリの画面が表示されることを確認します

### STEP 2：メニューを開く

画面右上の「**︙**」（縦3点）をタップします。

### STEP 3：ホーム画面に追加

1. メニューから「**ホーム画面に追加**」をタップします

2. アイコンの名前を確認・変更します
   - おすすめ：「お守り管理」「在庫管理」など

3. 「**追加**」をタップします

4. 確認ダイアログが出たら「**追加**」をタップします

### STEP 4：完了！

ホーム画面にアイコンが追加されます。
以降は、このアイコンをタップするだけでアプリが開きます。

## QRコードで簡単にURLを共有する

スタッフにURLを伝える際、QRコードを使うと便利です。

### QRコードの作成方法

1. 以下のサイトにアクセス：
   ```
   https://www.qrcode-monkey.com/
   ```

2. 「Your URL」欄にアプリのURLを入力
   ```
   https://[ユーザー名].github.io/omamori-app/
   ```

3. 「Create QR Code」をクリック

4. 「Download PNG」でダウンロード

5. このQRコードを印刷して休憩室などに掲示

### スタッフはQRコードを読み取るだけ

1. スマホのカメラでQRコードを読み取る
2. 表示されたURLをタップ
3. 上記の「ホーム画面に追加」手順を実行

## よくある質問

### Q: Safari以外のブラウザでもホーム画面に追加できますか？

**A:** iPhoneの場合、ホーム画面への追加はSafariでのみ可能です。Chromeなど他のブラウザでは「ホーム画面に追加」の選択肢が表示されません。

### Q: ホーム画面に追加したアイコンを削除するには？

**A:** 通常のアプリと同じように、アイコンを長押しして「削除」を選択します。

### Q: オフラインでも使えますか？

**A:** 現在のバージョンでは、インターネット接続が必要です。ただし、一度ページを開いた後は、ブラウザのキャッシュにより短時間であればオフラインでも動作する場合があります。

### Q: 機種変更したらどうなりますか？

**A:** 新しいスマホで同じ手順（URLを開いてホーム画面に追加）を行えば、すぐに使えるようになります。データはスマホに保存されないため、移行作業は不要です。

### Q: URLが長くて覚えられません

**A:** QRコードを作成して掲示するか、短縮URLサービス（bit.lyなど）を使うと便利です。

## トラブルシューティング

### 「ホーム画面に追加」が表示されない

| 原因 | 対処法 |
|------|--------|
| Safari以外のブラウザを使っている（iPhone） | Safariで開き直す |
| 共有メニューの下の方にある | 下にスクロールして探す |
| アクションが非表示になっている | 「アクションを編集」で有効化 |

### アイコンをタップしても開かない

| 原因 | 対処法 |
|------|--------|
| インターネットに接続されていない | Wi-Fiまたはモバイルデータを確認 |
| URLが変更された | 管理者に新しいURLを確認 |

### カメラが起動しない

| 原因 | 対処法 |
|------|--------|
| カメラの権限が許可されていない | 設定アプリ > Safari > カメラ を許可 |
| 古いiOS/Android | OSを最新版にアップデート |

## セキュリティについて

このアプリは以下の点で安全に設計されています：

| 項目 | 説明 |
|------|------|
| データの保存場所 | スマホ内のみ（サーバーに送信しない） |
| ログイン | 不要（個人情報を扱わない） |
| 通信の暗号化 | HTTPS（GitHub Pages標準） |

スキャンしたデータはCSV出力するまでスマホ内にのみ存在し、ブラウザを閉じると消えます。そのため、情報漏洩のリスクは極めて低いです。
