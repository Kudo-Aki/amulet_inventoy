# お守り在庫管理アプリ 簡易マニュアル

## はじめに

このアプリは、お守りの箱に貼られたQRコードをスマホで読み取って、在庫を数えるためのものです。

**ポイント**
- 同じ箱を2回読み取っても、数は増えません（安心して使えます）
- 読み取った結果はCSVファイルで出力できます

---

## 準備するもの

1. **スマホ**（iPhone または Android）
2. **QRコードが貼られたお守りの箱**

---

## 基本的な使い方

### ステップ1: アプリを開く

1. スマホのブラウザ（Safari や Chrome）でアプリを開きます
2. 「お守り在庫管理」の画面が表示されます

### ステップ2: モードを選ぶ

| モード | いつ使う？ |
|--------|-----------|
| 📦 **納品モード** | 新しいお守りが届いたとき |
| 📋 **棚卸モード** | 今ある在庫を数えるとき |

1. 使いたいモードのボタンを押します
2. 確認画面が出るので「開始する」を押します

### ステップ3: QRコードを読み取る

1. カメラが起動します
2. お守りの箱のQRコードをカメラに映します
3. 読み取りに成功すると、商品名が表示されます

**読み取りのコツ**
- QRコードを画面の真ん中に映す
- 明るい場所で読み取る
- カメラを近づけすぎない（20〜30cm程度）

### ステップ4: 読み取りを終了する

1. すべての箱を読み取ったら「読み取り終了」ボタンを押します
2. 集計結果が表示されます

### ステップ5: CSVを出力する

1. 「CSV出力」ボタンを押します
2. ファイルがダウンロードされます
3. このファイルをExcelで開くと、Airレジへの入力に使えます

---

## 同じ箱を2回読み取ってしまったら？

**大丈夫です！**

同じQRコードを2回読み取ると、以下のような警告が表示されます：

```
⚠️ 重複検知

この箱はすでに登録されています。
数量は増えません。
```

「OK」を押して、次の箱を読み取ってください。

## よくある質問

### Q: カメラが起動しない

**A:** ブラウザがカメラを使う許可を求めてきたら「許可」を押してください。

### Q: QRコードが読み取れない

**A:** 以下を試してください：
- 明るい場所に移動する
- QRコードをまっすぐカメラに向ける
- 少し離れて読み取る

### Q: 途中でアプリを閉じてしまった

**A:** 残念ながら、読み取りデータは消えてしまいます。最初からやり直してください。

### Q: 読み取った数が合わない

**A:** 以下を確認してください：
- 読み取り忘れた箱がないか
- 同じ商品の別の箱を読み取ったか（同じ箱を2回読んでも増えません）

## QRコードの見方

QRコードには以下の情報が入っています：

```
HEALTH-25-001
  │     │   │
  │     │   └── 箱の番号（001〜999）
  │     └────── 年度（25 = 2025年）
  └──────────── 商品コード
```

**商品コード一覧**

| コード | 商品名 |
|--------|--------|
| HEALTH | 健康守り |
| MONEY | 金運守り |
| LOVE | 縁結び守り |
| TRAFFIC | 交通安全守り |
| STUDY | 学業成就守り |
| FAMILY | 家内安全守り |
| BUSINESS | 商売繁盛守り |
| CHILD | 子授け守り |
| RECOVERY | 病気平癒守り |
| LUCK | 開運守り |
